Front-matter

Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量。以下是预先定义的参数，您可以在模板中使用这些参数值：

参数 描述 默认值
layout 布局 post
title 标题
date 建立日期 文件建立日期
updated 更新日期 文件更新日期
comments 开启文章的评论功能 true
tags 标签（不适用于分页）
categories 分类（不适用于分页）
toc_style_simple 使用更简洁的toc风格 false
permalink 覆盖文章网址
excerpt 纯文本的页面摘要
disableNunjucks 启用渲染引擎 false
lang 设置语言 默认语言
sticky 文章置顶 0
cover 文章封面
comments 显示文章评论模块 true
description 页面描述
toc 是否显示文章目录 false
keywords 页面关键词
copyright 显示文章版权模块 true
mathjax 显示mathjax false
katex 显示katex false
aside 显示侧边栏 true
aplayer 在该页面下加载aplayer的js和css false
highlight_shrink 代码框是否展开 false
copyright_author: xxxx
copyright_author_href: https://xxxxxx.com
copyright_url: https://xxxxxx.com
copyright_info: 此文章版權歸 xxxxx 所有，如有轉載，請註明來自原作者
